# 数据结构

::: tip 常见数据结构

- 一、线性表

顺序线性表、链式线性表

- 二、队列

- 三、堆

大顶堆、小顶堆

- 四、栈

- 五、树

    - 二叉树、满二叉树、完全二叉树、二叉查找树、平衡二叉树、红黑树

    - 哈夫曼树、哈夫曼编码
    - B-树、B+树

- 六、散列表（哈希表）

- 七、跳跃表

- 八、图

    - 无向图、有向图、简单图、完全图、有向完全图
    - 深度优先搜索（Depth First Search，DFS）、广度优先搜索（Breadth First Search，BFS）

:::
